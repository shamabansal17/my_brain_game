💡 How to Solve Sudoku
The Rules
One of each number per row, column and box
Every row, column and 3x3 box must contain digits 1-9 exactly once. No repeats anywhere.
Tip 1 - Elimination
Cross off numbers that cannot go there
Pick a blank cell. Look at its row, column and box. Any number already there is eliminated. If only one remains, fill it in!
Example: A row already has 1-8, so the blank must be 9.
Tip 2 - Scanning
Find where a number must go in a box
Pick a digit. Look at a 3x3 box - which cells can it go in? If only one cell is possible, place it there.
Focus on numbers that appear most on the board - they are easiest to place.
Tip 3 - When does it show an error?
Only when a group is completely filled
Errors only appear when you fill the last empty cell in a row, column or box and the values are wrong. The whole group blinks red. No judgment while you are still filling in!
You can correct any number at any time - just click and retype.
